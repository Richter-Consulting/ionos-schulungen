# Installation von Git

Ob git bereits auf Ihrem System installiert ist, kÃ¶nnen Sie mit folgenden befehl in der Commandozeile / Terminal / Bash ausprobieren:

```bash
# Linux / macOS
$ git --version
git version 2.25.1 
```

```ps
# Windows
> git --version
git version 2.33.0.windows.2
```

Wir statt einer Versionsangabe eine Fehlermeldung angezeigt, dass das `git` Befehl nicht gefunden wurde, muss git zuerst installiert werden.
